Website Site Plan

WDD 131 Project Planning Document

1. Site Name

Chosen Name: EcoTrails Explorer

Reason for selection: The name "EcoTrails Explorer" clearly communicates the site's primary theme: eco-friendly hiking and outdoor trail exploration.

2. Site Purpose

The purpose of EcoTrails Explorer is to provide a curated hub for regional sustainable hiking trails, detailed eco-tips for hikers, and a platform where outdoor enthusiasts can find reliable map checklists.

3. Target Audience Scenarios

  • Scenario 1: "What are the best beginner-friendly hiking trails in my state that emphasize minimal environmental footprint?"
  • Scenario 2: "Where can I find an actionable checklist of sustainable hiking gear before heading out?"

4. Color Scheme

5. Typography

  • Body Font: 'Open Sans', sans-serif
